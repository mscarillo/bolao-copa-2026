Como configurar o Supabase (grátis):
1. Acesse supabase.com → crie conta → New Project
2. Aguarde o projeto inicializar (~1 min)
3. Vá em SQL Editor e execute o script abaixo para criar as tabelas:
CREATE TABLE IF NOT EXISTS palpites ( id BIGSERIAL PRIMARY KEY, nome TEXT NOT NULL, contato TEXT, jogos JSONB NOT NULL DEFAULT '{}', bonus JSONB NOT NULL DEFAULT '{}', atualizado_em TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS resultados ( jogo_num INT PRIMARY KEY, time1 TEXT, gols1 INT, time2 TEXT, gols2 INT, atualizado_em TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE palpites ENABLE ROW LEVEL SECURITY; ALTER TABLE resultados ENABLE ROW LEVEL SECURITY; CREATE POLICY "public read palpites" ON palpites FOR SELECT USING (true); CREATE POLICY "public read resultados" ON resultados FOR SELECT USING (true); CREATE POLICY "anon insert palpites" ON palpites FOR INSERT WITH CHECK (true); CREATE POLICY "anon upsert resultados" ON resultados FOR ALL USING (true);
4. Vá em Project Settings → API
5. Copie a Project URL e a anon/public key
6. Clique em "Configurar Supabase" acima e cole as credenciais
