📄 Como atualizar os resultados

    

      O site lê o arquivo resultados.json do seu repositório GitHub. Para atualizar os placares, basta editar esse arquivo diretamente pelo site do GitHub — sem precisar de nada instalado.
    

    

      

        Passo 1 — Crie o arquivo na primeira vez

        No repositório GitHub: Add file → Create new file

        Nome: resultados.json

        Cole o conteúdo do modelo abaixo e clique Commit changes.
      

      

        Passo 2 — Atualizar após cada jogo

        No repositório GitHub: clique em resultados.json → lápis (editar)

        Adicione o placar do jogo na lista e clique Commit changes.

        O site atualiza em segundos ao clicar "Buscar Resultados".
      

      

        Passo 3 — Configure seu usuário no index.html

        Procure no arquivo: const GITHUB_USER = 'SEU_USUARIO';

        Troque SEU_USUARIO pelo seu usuário real do GitHub.
